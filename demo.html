Quick Start

Add this script to your website to embed the widget:

<script src="http://localhost:5173/widget.js"></script> <script> window.initGuidanceWidget({ primaryColor: '#9333ea', secondaryColor: '#f1f5f9', fontFamily: 'Inter, sans-serif', fontSize: '16px', logoUrl: '/your-logo.png' }); </script>

Configuration Options

primaryColor

Main brand color for buttons and highlights

primaryColor: '#9333ea'

secondaryColor

Background color for secondary elements

secondaryColor: '#f1f5f9'

fontFamily

Font family for all text in the widget

fontFamily: 'Inter, sans-serif'

fontSize

Base font size for the widget

fontSize: '16px'

logoUrl

URL to your company logo (optional)

logoUrl: '/your-logo.png'

maxQuestionsBeforeSolution

Number of questions before showing solution

maxQuestionsBeforeSolution: 3

Auto-Initialization

You can also auto-initialize the widget using data attributes:

<script src="http://localhost:5173/widget.js" data-auto-init="true" data-primary-color="#9333ea" data-secondary-color="#f1f5f9" data-font-family="Inter, sans-serif" data-font-size="16px" data-logo-url="/your-logo.png" ></script>
